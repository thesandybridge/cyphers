# Cypher CLI

A collection of cyphers CLI built with rust. Build the binary and execute.

# Installation

## Linux

The downloaded file will be located in `~/.local/bin/`

```bash
curl -s https://tinyurl.com/thesandybridge-cypher | bash
```

# Basic Usage

```shell
$ ./cypher rot "Hello, world!"
"Uryyb, Jbeyq!"
```
